Debugging

Trace Options

The following options can be passed to tvheadend to provide detailed debugging information while the application is running. Trace debugging has to be enabled at build time (--enable-debugging) and can then either be specified on the command- line (e.g. tvheadend -u hts -g video --trace <module>) or in the web interface (Configuration -> Debugging).

More than one option can be used, e.g. –trace cwc,dvr,linuxdvb
